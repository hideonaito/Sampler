# Sampler

Sampler
サンプラー

サンプラーは、録音した音（サンプル）を取り込み、切り貼りやピッチ（音程）の変更、エフェクト加工などを行い、鍵盤やパッドで自由に再生・演奏できる電子楽器のことで、音楽制作からライブ、動画配信まで幅広いシーンで活躍しています。

詳細は下記参照
https://ja.wikipedia.org/wiki/%E3%82%B5%E3%83%B3%E3%83%97%E3%83%A9%E3%83%BC

そのサンプラーのアプリを作成したい。
インターフェイスは「AKAI MPC2000」など過去の名機へのオマージュを込めたデザインにする。

サンプラーで実現したいことは下記の「基本機能」を参照。

＜基本機能＞
※まずは①②③の機能のみで作成
①サンプリング: 楽器の音や人間の声、環境音、既存の楽曲の一部などを録音・取り込む機能。
②再生：鍵盤やパッドにサンプリング音源を割り当て、出力する機能。
③エフェクト：「PitchShifter」「Reverb」「Delay」など録音されたサンプリング音源を加工する機能。
④スライス: 取り込んだサンプリング音源を細かく分割し、それぞれの断片を別々のパッドに割り当てる機能。
⑤リサンプリング: エフェクトをかけたり、演奏したフレーズを、新しいサンプリング音源として録音する機能。
⑥シーケンサー: 録音した音をタイミングよく並べて、自動演奏やリズムパターン（ビート）を構築する機能。
